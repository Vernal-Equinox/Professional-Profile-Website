Jet Game

A simple 2D jet-flight game built in C++ using OpenGL for rendering.

About the Project

This is a simple 2D Jet Game where players control a jet fighter navigating through a series of enemy. The game is developed using C++ for the core logic and OpenGL for rendering the 2D graphics. In this game, players are tasked with avoiding the enemy, dodging incoming enemy fire, and collecting power-ups to enhance their jet’s abilities. The game features simple yet engaging mechanics, smooth animations, and responsive controls.

The system is designed to be lightweight, optimized for performance, and capable of running on most devices that support OpenGL. The simplicity of the 2D environment allows the focus to be on smooth gameplay, quick reflexes, and achieving high scores.

Key Features

Smooth 2D Jet Control

Players can easily control their jet using the arrow keys to navigate within the game space.

OpenGL-Based 2D Rendering

Graphics are rendered using OpenGL for fast and efficient 2D drawing, providing smooth animations, crisp textures, and responsive movement. All the game objects used are represented with basic 2d sprites.

Power-Ups and Collectibles

Players can collect power-ups that enhance the jet's abilities.

Score Tracking

The game tracks the player’s score based on the number of enemies destroyed. Players can attempt to achieve high scores, adding replay value.
